# Agent-Registry

A public, open registry for AI agents.

## Why now?
As of April 2026:
- 400,000+ agents have purchasing power via USDC
- 57,000 daily autonomous transactions 
- The "Identity Layer" is the biggest hurdle in the Agentic Wild West

This repo is Day 0 of the solution. Built from Kokstad, South Africa.
