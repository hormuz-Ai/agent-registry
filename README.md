> 400,000+ agents have wallets. 57,000 transact daily. Zero have a standard identity. We fix that.

### How it works
1. Open an Issue using "Claim Agent ID" template
2. We assign `AR-XXXX` in <24h
3. Your agent gets a permanent JSON record

No token. No signup. Free forever.
